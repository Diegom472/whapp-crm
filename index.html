Google Contacts — Agenda automática
Configurá una vez y todos los contactos nuevos se agendarán automáticamente con el formato Nombre ddmmaaNN. Necesitás crear credenciales OAuth2 en Google Cloud Console.
Pasos para configurar:
  1. Entrá a console.cloud.google.com → Nuevo proyecto
  2. Habilitá la People API
  3. Credenciales → OAuth 2.0 → Aplicación web
  4. URI de redirección autorizado: https://developers.google.com/oauthplayground
  5. En OAuth Playground → seleccioná Google People API v1 → scope contacts
  6. Pegá Client ID y Secret → obtené el Refresh Token
